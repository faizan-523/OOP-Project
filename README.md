# ✈️ Flight Management System (OOP Project – 2nd Semester BSCS)

## 📖 Project Overview
This project is a **Flight Management System** implemented in **C++** using **Object-Oriented Programming (OOP)** concepts.  
It is designed to manage **flights, passengers, and staff**, demonstrating the use of **classes, inheritance, file handling, and OOP principles**.  

This project was developed as part of the **2nd semester BSCS curriculum** to apply theoretical OOP knowledge in a practical console-based application.

---

## 🎯 Objectives
The main objectives of this project are:

- To practice **Object-Oriented Programming** concepts such as classes, objects, inheritance, static members, and friend functions  
- To implement **data management** for flights, passengers, and staff  
- To practice **file handling in C++** for storing and retrieving data  
- To create a **functional console-based system** simulating a basic airline management application  

---

## 🛠 Technologies Used
- **C++** – Programming language  
- **OOP Concepts** – Classes, Inheritance, Friend functions, Static members  
- **File Handling** – Text files to store flight, passenger, and staff information

---

## 💡 Features
- Add, update, and remove **flights**  
- Add, update, and remove **passengers**  
- Add, update, and remove **staff members**  
- Persistent storage using **text files**  
- Console-based menu for easy interaction
